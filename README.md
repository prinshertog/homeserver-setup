# Media Setup Ansible

## BELANGRIJK

Dit script is alleen getest op debian 13 (trixie)

## Benodigdheden

- python (voor ansible)
- ansible-playbook

```
python3 -m pip install --user ansible
```

## Uitvoeren van het script
```
./run_ansible.sh
```
